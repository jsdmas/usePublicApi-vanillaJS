# 📉 서울시 행사정보 조회

공공 데이터 포탈의 API, kakao map API 외부 API를 활용해 웹 페이지를 구현한 프로젝트입니다.

## 📚 실행방법

```
npm start
```

# Front-End skills
- vanilla javaScript, scss, webpack
